Cet atelier consiste à créer un projet Python pour gérer les voitures de service d’une entreprise.
Le projet comprend deux classes : Employé et Voiture, avec leurs attributs et méthodes.
Les fonctionnalités incluent :
Afficher les informations de tous les objets créés.
Affecter des voitures à des employés et mettre à jour les informations.
Retirer certaines voitures d’employés et afficher les changements.
Gérer les cas où une voiture est déjà affectée à un autre employé.
